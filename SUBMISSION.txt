-------------------------------------------
DATABASE SCHEMA
-------------------------------------------

Table 1: Users
- id (PRIMARY KEY)
- name
- email (UNIQUE)
- password (hashed)
- created_at

Table 2: Subjects
- id (PRIMARY KEY)
- user_id (FOREIGN KEY → Users)
- name
- description
- created_at

Table 3: StudySessions
- id (PRIMARY KEY)
- user_id (FOREIGN KEY → Users)
- subject_id (FOREIGN KEY → Subjects)
- title
- description
- start_time
- end_time
- status (pending/completed/cancelled)
- created_at
